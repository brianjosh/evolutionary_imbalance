# evolutionary_imbalance

This provides the code and data files necessary to repeat the analyses described in 'Plants that evolved under high phylogenetic diversity have higher invasion success, particularly in undisturbed communities' (Brian, van Kleunen, Dawson, Kempel, Zhao, Catford). Analyses and usage of all other files is carried out in 'Brian_etal_code_final.R', except for 'Variable_table.docx', which provides details on all explanatory and response variables used in 'Brian_etal_code_final.R'. Questions should be directed to Joshua Brian (jshbrian@gmail.com).
